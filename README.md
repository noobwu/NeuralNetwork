## 什么是神经元？

"神经元是神经网络中计算的基本单位。"

正如前面提到的，神经元有时也被称为节点或单元。它接收来自其他节点或外部源的输入并计算输出。每个输入都有一个相关的**权重**（见图1—1中的w1和w2） ，它是根据输入信息的相对重要性进行分配的。节点将输入的加权和提供给函数 （**激活函数**，后面将详细介绍）。虽然这是对神经元是什么以及它能做什么的极端简化，但基本上就是这样。
